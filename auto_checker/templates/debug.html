Upload one or more PDF answer sheets. The printed/typed text is treated as the question (buggy pseudocode). The handwritten text is treated as the student's answer. Student name should be on the first page. Supports 2 question paper sets.

How It Works

  1. Upload student answer sheet PDFs.
  2. AI reads the sheet — typed/printed text is the buggy pseudocode question.
  3. Handwritten text is the student's identified error and corrected output.
  4. AI evaluates if the student correctly found the error and wrote the right output.

Scoring Rules

  • Correct error identification: 1 point
  • Correct output: 2 points
  • Total per question: 3 points
